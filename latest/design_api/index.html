Note

All fields of structs can be accessed as functions.

PCB

PlanerTransformer.Stackup — Type.
Stackup(material,thickness)

Specify the PCB stackup.

example: for double sided FR408 PCB with 1oz copper 63mil thick. Stackup([copper,fr408,copper],[0.48e-3,1.6e-3 - (2*0.48e-3),0.48e-3])

copper and fr408 are exported constants.

PlanerTransformer.copper_weight_to_meters — Function.
copper_weight_to_meters(oz)

PCB copper thickness is typicaly given in ounces. This function multiplies by 35e-6 to give thickness in meters.

Fields

Windings

PlanerTransformer.Windings — Type.
Windings

Fields

  • pcb – PCB_Specification

  • core – CoreGeometry

  • primarywindinggeometry – WindingGeometry for all layers of primary

  • secondarywindinggeometry– WindingGeometry for all layers of secondary

  • isprimary – tuple of Bool for each conductor layer

  • sides – for internal use

  • isprimaryseries – true if primary is connected in series

  • issecondaryseries – true if secondary is connected in series

Create Windings objects with windings.

Analysis

PlanerTransformer.TransformerPowerAnalysis — Type.
TransformerPowerAnalysis

Object to store the result of a transformer_power_analysis().

Fields

  • transformer – from input

  • converter – type of converter.

  • flux_density – peak flux density in Tesla

  • voltage – (primary, secondary) Vpp

  • current – (promary, secondary) Ipp

  • ac_winding_resistance – (primary, secondary) Ω

  • dc_winding_resistance – (primary, secondary) Ω

  • equilivent_resistance – (primary, secondary) Ω

  • v_core – voltage, after drop accross input resistance, referenced to 1 turn

  • core_specific_power – power dissipated in core in W/m^3

  • core_total_power – power dissipated in core in W
