    Introduction

    Introduction

    This module is based on the design procedure described in Designing Planar Magnetics.

    Load the module.

    using PlanerTransformer

    Define the PCB.

    trace_edge_gap = 0.16e-3
trace_trace_gap = 0.13e-3
outer_copper_thickness = copper_weight_to_meters(1.0)
inner_copper_thickness = copper_weight_to_meters(1.0)
number_of_layers = 4
pcb = PCB_Specification(trace_edge_gap,
                        trace_trace_gap,
                        outer_copper_thickness,
                        inner_copper_thickness,
                        number_of_layers)
    PlanerTransformer.PCB_Specification(0.00016, 0.00013, 0.00048, 0.00048, 4, 1.68e-8, 0.003862)

    Operate at 500kHz with a maximum specific power loss of 150kW/m^3.

    frequency = 0.5e6 # Hz
spl_max = 150e3 # W/m^3
    150000.0

    Define core geometry and type of ferrite, or use data from core_geometry_dict and ferrite_dict.

    core = core_geometry_dict["er25/5.5_plt"]
    PlanerTransformer.CoreGeometry("er25/5.5_plt", 0.0055, 0.0055, 0.0, 2.37e-6, 8.97e-5, 0.0264, 0.0131)
    material = ferrite_dict["3f35"]
    PlanerTransformer.FerriteProperties("3F35", 500000.0, 1.0e6, 25.0, 100.0, PlanerTransformer.BHloop(40.0, 0.45, 0.2), PlanerTransformer.BHloop(25.0, 0.4, 0.15), PlanerTransformer.SpecificPowerLossData((500000.0, 1.0e6), ((2.91449357768842, 8.728534444848094), (2.447942813807877, 9.004079453003751))))

    Define the magnetics as a material and a core geometry.

    magnetics = Magnetics(material, core)
    PlanerTransformer.Magnetics(PlanerTransformer.FerriteProperties("3F35", 500000.0, 1.0e6, 25.0, 100.0, PlanerTransformer.BHloop(40.0, 0.45, 0.2), PlanerTransformer.BHloop(25.0, 0.4, 0.15), PlanerTransformer.SpecificPowerLossData((500000.0, 1.0e6), ((2.91449357768842, 8.728534444848094), (2.447942813807877, 9.004079453003751)))), PlanerTransformer.CoreGeometry("er25/5.5_plt", 0.0055, 0.0055, 0.0, 2.37e-6, 8.97e-5, 0.0264, 0.0131))

    Interpolate off of specific power loss data to maximum flux density.

    flux_density_max_pp = flux_density(magnetics, spl_max, frequency) * 2.0
    0.12082069532040642

    Calculate maximum volt seconds per turn.

    vspt = volt_seconds_per_turn(magnetics, flux_density_max_pp)
    1.0837616370240456e-5

    Define the windings. These windings are just given as an example use of the package. They are not optimized for any input voltage and output current. The primary will be 3 turns, 3 on each outer layer, both layers in parallel. The secondary will be 4 turns, 2 turns on each inner layer, both layers in series.

    layer1 = winding_layer(pcb, true, core, 3)
layer2 = winding_layer(pcb, false, core, 2)
layer3 = winding_layer(pcb, false, core, 2)
layer4 = winding_layer(pcb, true, core, 3)
primary = Winding(pcb, [layer1, layer4], false)
secondary = Winding(pcb, [layer2, layer3], true)
    PlanerTransformer.Winding(PlanerTransformer.PCB_Specification(0.00016, 0.00013, 0.00048, 0.00048, 4, 1.68e-8, 0.003862), PlanerTransformer.WindingLayer[PlanerTransformer.WindingLayer(0.002525, 0.103673, 0.00048, 2), PlanerTransformer.WindingLayer(0.002525, 0.103673, 0.00048, 2)], true, 4)

    Define the transformer. The first element in the winding array should be the input to the transformer.

    transformer = Transformer(magnetics, [primary, secondary])
    PlanerTransformer.Transformer(PlanerTransformer.Magnetics(PlanerTransformer.FerriteProperties("3F35", 500000.0, 1.0e6, 25.0, 100.0, PlanerTransformer.BHloop(40.0, 0.45, 0.2), PlanerTransformer.BHloop(25.0, 0.4, 0.15), PlanerTransformer.SpecificPowerLossData((500000.0, 1.0e6), ((2.91449357768842, 8.728534444848094), (2.447942813807877, 9.004079453003751)))), PlanerTransformer.CoreGeometry("er25/5.5_plt", 0.0055, 0.0055, 0.0, 2.37e-6, 8.97e-5, 0.0264, 0.0131)), PlanerTransformer.Winding[PlanerTransformer.Winding(PlanerTransformer.PCB_Specification(0.00016, 0.00013, 0.00048, 0.00048, 4, 1.68e-8, 0.003862), PlanerTransformer.WindingLayer[PlanerTransformer.WindingLayer(0.00164, 0.155509, 0.00048, 3), PlanerTransformer.WindingLayer(0.00164, 0.155509, 0.00048, 3)], false, 3), PlanerTransformer.Winding(PlanerTransformer.PCB_Specification(0.00016, 0.00013, 0.00048, 0.00048, 4, 1.68e-8, 0.003862), PlanerTransformer.WindingLayer[PlanerTransformer.WindingLayer(0.002525, 0.103673, 0.00048, 2), PlanerTransformer.WindingLayer(0.002525, 0.103673, 0.00048, 2)], true, 4)])

    Now that the transformer is defined, we can estimate some circuit parameters.

    Maximum volt seconds on each winding.

    volt_seconds_max = volt_seconds(transformer, flux_density_max_pp)
    2-element Array{Float64,1}:
 3.25128e-5
 4.33505e-5

    Power dissipation with 0.5Vpp 50% duty cycle square wave on the primary and 1App load on the secondary.

    primary_voltage_pp = 0.5
output_current_pp = 1.0
tpd =TransformerPowerDissipation(transformer,
              [primary_voltage_pp, output_current_pp], frequency)
tp = total_power(tpd) # W
    0.007715285755142156
